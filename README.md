# 🎬 Movies App

A responsive movie discovery application built with React and Vite, 
allowing users to search and explore movies using real-time data.

## 🚀 Live Demo
[View Live](#) <!-- Add your Vercel/Netlify link here -->

## ✨ Features
- 🔍 Search movies in real-time
- 🎥 Browse popular and trending movies
- 📱 Fully responsive design
- ⚡ Fast performance with Vite bundling

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript (ES6+), CSS
- **Build Tool:** Vite
- **API:** TMDB / OMDB API
- **Deployment:** Vercel
