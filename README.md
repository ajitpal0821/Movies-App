# 🎬 Movies App

A responsive movie discovery application built with React and Vite, 
allowing users to search and explore movies using real-time data.

## 🚀 Live Demo
https://ajitpal0821.github.io/Movies-App<!-- Add your Vercel/Netlify link here -->

## ✨ Features
- 🔍 Search movies in real-time
- 🎥 Browse popular and trending movies
- 📱 Fully responsive design
- ⚡ Fast performance with Vite bundling

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript (ES6+), CSS
- **Build Tool:** Vite
- **API:** TMDB / OMDB API
- **Deployment:** GitHub Pages

## 📦 Installation
```bash
# Clone the repository
git clone https://github.com/ajitpal0821/Movies-App.git

# Navigate to project directory
cd Movies-App

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🔑 Environment Variables
Create a `.env` file in root directory:
```
VITE_API_KEY=your_api_key_here
```

## 📁 Project Structure
```
src/
├── components/      # Reusable UI components
├── pages/           # Page components
├── services/        # API calls and external service logic
├── css/             # Global and componenstyles
└── App.jsx          # Root component
```
